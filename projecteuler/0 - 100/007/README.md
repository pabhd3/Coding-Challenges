# [Nth Prime](https://projecteuler.net/problem=7)

### Problem 007

By listing the first six prime numbers: 2, 3, 5, 7, 11, and 13, we can see that the 6th prime is 13.

What is the *nth* prime number?

---

## How to Use

#### **PHP**

* Run `php nthPrime.php` from the command line.
* Enter an integer, *n*, and hit enter.
* **Note n > 5,000 will take longer to calculate.**

#### **Python3**

* Run `python3 nthPrime.py` from the command line.
* Enter an integer, *n*, and hit enter.
* **Note n > 5,000 will take longer to calculate.**

#### **Scala**

* Run `scala nthPrime.scala n` from the command line.
* Set *n* equal to an integer.
* **Note n > 5,000 will take longer to calculate.**
